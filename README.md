## Usage

1. Right-click on any folder in the Explorer
2. Select **"Colorize"**
3. Pick a color from the palette, or select **"Clear color"** to remove

## Color Palette

The palette includes carefully selected pastel colors organized by hue:

| Category | Colors |
|----------|--------|

| **Grays** | 8 shades from Dark Gray to Very Light Gray |

## Credits

This extension is based on [Folder Color](https://github.com/egorovsa/vscode-folder-colorizer) by Sergey Egorov.

### Changes from the original:
- Curated pastel color palette (32 colors instead of 200+)
- Colors sorted by hue for easier selection
- Unified brightness for consistent appearance
- Simplified context menu (removed submenu)
- Shows current color in the "Clear color" option
- Runtime SVG icon generation (no pre-generated icon files)
- Removed badge/emoji functionality (focused on colors only)
- New extension icon
